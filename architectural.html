Architectural design and development

I am currently studying Middle management officer construction at MBO college Hilversum, learning the designing of constructions and details, orginizing and leading build projects and creating drawings, work descriptions, schedules and more.

Autocad

Autocad is a program used for architectural drawings and floor plans for buildings. In my first year of education I learned to work with the program and make architectural drawings. The project shown on the left was my final project for the first year of college. We were given a concept home to expand on. I started by creating the floor plans and facade of the building. I learned how to draw the foundation, various details, intersection and more. This project helped me learn a lot about the progam. I am proficient at Autocad and know my way around the program.

Revit

Revit is used to create a more construction based 3D model for a building. But can also be used like Autocad for a 2D floor plan. I have only been using this program since my second year of college, this has not stopped me from pick up the program rather quick and I would call myself experienced. The school project shown on the right is a specification drawing of the floor plans and 3D model of a terranced house. I started creating the first floor and worked my way up adding furniture in specific areas like the kitchen and bathroom. In a Revit 3D model you are able to go more into dept with the material used like isolation, foundation and the walls.

Sketchup

Sketchup is a 3D drawing program used for making 3D models and structural visuals. I have been using Sketchup since 2018 and have been using it to make sketches of buildings for my school and personal projects. I am very experienced with the program.

Inventor

Inventor is a mechanical engineering program mainly used to create metal framework or mechanical components. Having been able to learn the program while interning at PLEKvoor, I have built up a basic understanding of the program.
